Analisis Plot Pelatihan

Di bawah ini adalah perbandingan hasil pelatihan antara model **Baseline** (hanya dilatih pada data bersih) dan model **Robust** (dilatih dengan data augmentasi, termasuk noise). Plot ini sangat penting untuk memahami *overfitting*.

Model BASELINE

Penjelasan: Perhatikan! Garis 'Training Accuracy' (biru) terus naik, sementara 'Validation Accuracy' (oranye) stagnan/turun. Ini adalah tanda klasik **overfitting**. Model menjadi terlalu "hafal" data latih dan tidak bisa menggeneralisasi ke data baru (validasi).

Model ROBUST

Penjelasan: Dengan data augmentasi, kurva training dan validasi bergerak **lebih berdekatan**. Ini menunjukkan model tidak *overfitting*. Tugasnya menjadi lebih sulit (makanya akurasi latih lebih rendah), tetapi ia belajar fitur yang lebih umum (generalisasi), sehingga performa validasinya lebih baik dan stabil.
